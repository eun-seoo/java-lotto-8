# java-lotto-precourse

## 🎯 로또 (Lotto)

이 프로그램은 간단한 **로또 발매기**를 구현한 콘솔 기반 애플리케이션입니다.  
사용자는 구입 금액을 입력하면 해당 금액에 맞게 로또를 자동으로 발행할 수 있으며,  
발행된 로또 번호와 당첨 번호를 비교하여 **당첨 내역과 수익률**을 계산합니다.  
잘못된 입력이 들어올 경우 예외를 발생시키고, 사용자가 올바른 입력을 다시 시도할 수 있도록 합니다.

## 📋 기능 목록

### 입력
- 구입 금액을 입력받는다.
  - [예외] 1,000원 단위로 나누어 떨어지지 않으면 예외 발생
  - [예외] 음수나 문자가 포함되면 예외 발생
- 당첨 번호를 입력받는다. (쉼표로 구분)
  - [예외] 중복된 번호 입력 시 예외 발생
  - [예외] 1~45 범위를 벗어나는 경우 예외 발생
- 보너스 번호를 입력받는다.
  - [예외] 당첨 번호와 중복되면 예외 발생
  - [예외] 1~45 범위를 벗어나는 경우 예외 발생

### 로또 발행
- 구입 금액에 해당하는 개수만큼 로또를 발행한다.
- 로또 번호는 1~45 사이의 중복되지 않은 6개의 숫자로 구성된다.
- 발행된 로또 번호는 오름차순으로 정렬된다.

### 당첨 결과 계산
- 당첨 번호와 사용자의 로또를 비교하여 일치 개수를 계산한다.
- 등수별 당첨 금액을 합산하고 총 수익률을 계산한다.
- 수익률은 소수점 둘째 자리에서 반올림한다.

### 출력
- 구매한 로또 수량과 번호를 출력한다.
- 당첨 내역 및 수익률을 출력한다.
- [예외] 잘못된 입력 시 `[ERROR]`로 시작하는 에러 메시지를 출력한다.
